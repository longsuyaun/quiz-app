# 📱 错题自测系统 - 部署到公网

## 方案 1：GitHub Pages（推荐，免费）

### 步骤

1. **创建 GitHub 仓库**
   ```bash
   cd /home/bao/.openclaw/workspace/apps/quiz-app
   git init
   git add .
   git commit -m "Initial commit"
   ```

2. **推送到 GitHub**
   ```bash
   git remote add origin https://github.com/你的用户名/quiz-app.git
   git branch -M main
   git push -u origin main
   ```

3. **启用 GitHub Pages**
   - 打开 https://github.com/你的用户名/quiz-app/settings/pages
   - Source 选择 "main branch"
   - 保存

4. **访问地址**
   ```
   https://你的用户名.github.io/quiz-app/
   ```

### 优点
- ✅ 完全免费
- ✅ 自动 HTTPS
- ✅ 自定义域名
- ✅ 自动更新（push 即部署）

---

## 方案 2：Vercel（免费，更快）

### 步骤

1. **安装 Vercel CLI**
   ```bash
   npm install -g vercel
   ```

2. **部署**
   ```bash
   cd /home/bao/.openclaw/workspace/apps/quiz-app
   vercel --prod
   ```

3. **访问地址**
   ```
   https://quiz-app-xxx.vercel.app
   ```

### 优点
- ✅ 全球 CDN
- ✅ 自动 HTTPS
- ✅ 速度快
- ✅ 免费额度够用

---

## 方案 3：阿里云 OSS（低成本）

### 步骤

1. **创建 OSS Bucket**
   - 登录阿里云控制台
   - 创建 Bucket（选择公共读）
   - 开启静态网站托管

2. **上传文件**
   ```bash
   ossutil cp -r apps/quiz-app/ oss://你的 bucket/
   ```

3. **访问地址**
   ```
   http://你的 bucket.oss-cn-hangzhou.aliyuncs.com/
   ```

### 成本
- 存储费：约 0.1 元/GB/月
- 流量费：约 0.5 元/GB
- 适合国内访问

---

## 推荐选择

**国内用户**: 阿里云 OSS（速度快）
**海外用户**: GitHub Pages（免费）
**追求速度**: Vercel（全球 CDN）
