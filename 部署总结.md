## 💡 使用流程

### 日常使用
1. **访问系统**（在线版或本地版）
2. **选择用户**（左左/右右）
3. **启用变式题模式**（推荐）
4. **选择科目**
5. **开始答题**

### 添加新题
1. **编辑** `apps/feishu-quiz/quiz-full.html`
2. **在 questionBank 中添加题目**
3. **保存文件**
4. **自动同步**（文件监听模式）
   - 或手动执行：`./trigger-sync.sh`
5. **收到飞书通知**

### 启动监听服务
```bash
# 安装依赖
sudo apt install inotify-tools

# 启动监听
cd /home/bao/.openclaw/workspace/apps/quiz-app
./start-watcher.sh
```

## 📈 系统特色

### 1. 双系统架构
- **在线版**: GitHub Pages，公网访问
- **本地版**: HTTP Server，局域网访问
- **自动同步**: 文件监听，保存后 2 秒同步

### 2. 变式题系统
- **数字变式**: 改变题目系数
- **问法变式**: 正确→错误
- **场景变式**: 改变物理场景
- **目的**: 避免死记硬背，真正掌握

### 3. 自动通知
- **成功通知**: 包含访问链接
- **失败通知**: 包含排查建议
- **重试机制**: 最多 3 次
