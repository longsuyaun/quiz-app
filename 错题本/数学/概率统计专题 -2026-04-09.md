## 题目 1：正态分布与独立事件概率

**录入时间**: 2026-04-09 23:42  
**知识点**: 正态分布对称性、独立事件概率  
**难度**: ⭐⭐⭐ 中等  
**题型**: 填空题

---

### 📝 题目

某单位安装的摄像头使用寿命 X（单位：年）服从正态分布。

已知：
- P(X ≥ 2) = 0.8（寿命不少于 2 年的概率）
- P(X > 6) = 0.2（寿命多于 6 年的概率）

该单位安装了**两个**这样的摄像头，求**4 年内两个都能正常工作**的概率。

---

### ✅ 答案

**0.25**（或 **1/4**）

---

### 📖 详细解析

**1. 利用正态分布的对称性**

已知：
- P(X ≥ 2) = 0.8 → P(X < 2) = 0.2
- P(X > 6) = 0.2

因为 P(X < 2) = P(X > 6) = 0.2，说明 **2 和 6 关于对称轴对称**

**对称轴（均值μ）**：
```
μ = (2 + 6) / 2 = 4
```

**2. 求单个摄像头寿命≥4 年的概率**

因为正态分布关于均值μ=4 对称：
```
P(X ≥ 4) = 0.5
```

**3. 求两个摄像头都能工作 4 年的概率**

两个摄像头**独立工作**，概率相乘：
```
P(两个都≥4 年) = P(X ≥ 4) × P(X ≥ 4)
               = 0.5 × 0.5
               = 0.25
```

---

### 📌 核心考点

| 知识点 | 公式/性质 |
|--------|----------|
| 正态分布对称性 | P(X < μ-a) = P(X > μ+a) |
| 均值计算 | μ = (a + b) / 2（当 P(X<a)=P(X>b) 时） |
| 独立事件概率 | P(A 且 B) = P(A) × P(B) |

---

### 💡 解题技巧

1. **找对称轴**：当 P(X<a) = P(X>b) 时，μ = (a+b)/2
2. **利用对称性**：P(X ≥ μ) = 0.5
3. **独立事件**：两个独立事件同时发生，概率相乘

---

### ⚠️ 易错点

- ❌ 忘记正态分布的对称性
- ❌ 不会求均值μ
- ❌ 两个事件概率忘记相乘（写成 0.5）

## 题目 2：正态分布参数计算

**录入时间**: 2026-04-09 23:43  
**知识点**: 正态分布 N(μ,σ²)、3σ原则、概率计算  
**难度**: ⭐⭐⭐⭐ 较难  
**题型**: 选择题

---

### 📝 题目

已知随机变量 X 服从正态分布 N(100, 4)，若 P(m ≤ X ≤ 104) = 0.1359，则 m 等于（ ）

**选项**：
- A. 100
- B. 101
- C. 102
- D. 103

**附**：
- P(μ-σ ≤ X ≤ μ+σ) ≈ 0.6827
- P(μ-2σ ≤ X ≤ μ+2σ) ≈ 0.9545

---

### ✅ 答案

**C. 102**

---

### 📖 详细解析

**1. 确定正态分布参数**

X ~ N(100, 4)
- 均值 μ = 100
- 方差 σ² = 4 → **标准差 σ = 2**

**2. 分析已知概率**

P(m ≤ X ≤ 104) = 0.1359

观察附图数据：
```
P(μ-2σ ≤ X ≤ μ+2σ) = 0.9545
P(μ-σ ≤ X ≤ μ+σ) = 0.6827
```

**3. 利用对称性计算**

从 100 到 104 的距离：
```
104 - 100 = 4 = 2σ
```

所以 104 = μ + 2σ

**4. 求 P(μ+σ ≤ X ≤ μ+2σ)**

根据对称性：
```
P(μ+σ ≤ X ≤ μ+2σ) = ½ × [P(μ-2σ ≤ X ≤ μ+2σ) - P(μ-σ ≤ X ≤ μ+σ)]
                   = ½ × (0.9545 - 0.6827)
                   = ½ × 0.2718
                   = 0.1359
```

**5. 确定 m 的值**

因为 P(m ≤ X ≤ 104) = 0.1359，且 104 = μ + 2σ

所以：
```
m = μ + σ = 100 + 2 = 102
```

---

### 📌 核心考点

| 知识点 | 数值 |
|--------|------|
| 均值 μ | 100 |
| 标准差 σ | 2 |
| μ+σ | 102 |
| μ+2σ | 104 |
| P(μ+σ ≤ X ≤ μ+2σ) | 0.1359 |

---

### 💡 解题技巧

1. **识别参数**：N(μ, σ²) 中第二个参数是方差
2. **记住 3σ 原则**：
   - 1σ 范围：约 68.27%
   - 2σ 范围：约 95.45%
   - 3σ 范围：约 99.73%
3. **利用对称性**：正态分布关于μ对称

---

### ⚠️ 易错点

- ❌ 误认为 N(100, 4) 中 4 是标准差（实际是方差）
- ❌ 不会利用已知概率数据计算
- ❌ 搞错对称关系
