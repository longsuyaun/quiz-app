## 💡 使用建议

### 日常使用
- **本地版**: 快速测试、即时更新
- **在线版**: 分享链接、外网访问

### 更新流程
1. **本地修改** quiz-full.html

3. **等待部署**（1-2 分钟）
4. **验证** 两个版本都更新

## 🔧 故障排查

### 问题 1: 在线版未更新
**解决**: 
1. 检查 Git 推送是否成功
2. 等待 GitHub Pages 部署（1-2 分钟）
3. 强制刷新浏览器（Ctrl+Shift+R）

### 问题 2: 同步脚本失败
**解决**:
1. 检查网络连接
2. 检查 GitHub token 配置
3. 手动执行 Git 命令
