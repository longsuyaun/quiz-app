### 题1：青蒿素结构与性质（化学）

**题目：** 青蒿素是治疗疟疾的有效药物，主要从黄花蒿中提取，其结构简式和实验室制备的部分流程如图。下列说法不正确的是：

- A. 青蒿素的分子式为 C₁₅H₂₁O₅
- B. 青蒿素中含氧官能团有过氧键、醚键和酯基
- C. 干燥破碎能增加黄花蒿与乙醚的接触面积，提高萃取率
- D. "减压蒸馏"能减少青蒿素中过氧键的分解

**我的答案：** A（正确）

**解析：**
- A 错误：青蒿素分子式应为 **C₁₅H₂₂O₅**，题目写 C₁₅H₂₁O₅（少1个H）
- B 正确：确实有过氧键(-O-O-)、醚键(C-O-C)、酯基(内酯)
- C 正确：增大接触面积提高萃取率
- D 正确：减压蒸馏降低沸点，避免过氧键高温分解

**知识点：** 有机物分子式判断、官能团识别、萃取原理、减压蒸馏
**易错点：** 分子式数氢要小心；内酯环中的C=O是酯羰基，不是酮羰基

### 题2：有机物命名（化学）

**题目：** 下列命名正确的是？结构：CH₃-CH(CH₂CH₃)-CH₂-CH₃

- A. 2-乙基丁烷
- B. 3-甲基戊烷
- C. 2-甲基戊烷
- D. 3-乙基丁烷

**我的答案：** B（正确）

**解析：**
- 找最长碳链：不是横着4个碳，而是**5个碳**（主链戊烷）
- 编号从右往左（支链编号最小）：甲基在3号位
- "2-乙基丁烷"是典型陷阱——没找最长链

**知识点：** IUPAC命名规则、最长碳链原则
**易错点：** 不能只看"横向"链，要找最长连续碳链

### 题3：官能团判断（化学）

**题目：** 下列有关官能团的说法错误的是：

- A. CH₃OOC-环己烯-COOCH₃ 中的官能团为碳碳双键和酯基
- B. 某化合物中的含氧官能团为（酚）羟基和羰基
- C. 高分子膨胀剂中的官能团为羧基和羟基
- D. H₂C=CHCH₃ + Br₂（光照）→ H₂C=CHCH₂Br 中生成的官能团是溴原子

**我的答案：** C（正确）

**解析：**
- A 正确：环己烯上有碳碳双键，两侧是酯基(-COOCH₃)
- B 正确：含酚羟基和羰基
- C 错误：结构中是**醚键(-O-)和羟基**，没有羧基(-COOH)
- D 正确：光照下α-取代生成C-Br

**知识点：** 官能团识别、苯环结构特点（共轭大π键≠碳碳双键）
**易错点：** 
- 苯环不是碳碳双键（是共轭大π键体系）
- 醚键(-O-)和羧基(-COOH)结构完全不同
- 环己烯是有碳碳双键的，不是苯环
