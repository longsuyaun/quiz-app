## 🚀 使用方式

### 方式 1：上传后手动触发

**步骤**:
1. 修改 `quiz-full.html`
2. 上传新题目
3. 执行同步脚本

```bash
cd /home/bao/.openclaw/workspace/apps/quiz-app
./trigger-sync.sh
```

---

### 方式 2：配置快捷键（推荐）

**创建桌面快捷方式**:
```bash
# 创建桌面文件
cat > ~/Desktop/同步错题系统.desktop << EOF
[Desktop Entry]
Type=Application
Name=🔄 同步错题系统
Exec=/home/bao/.openclaw/workspace/apps/quiz-app/trigger-sync.sh
Icon=system-run
Terminal=true
EOF

chmod +x ~/Desktop/同步错题系统.desktop
```

**双击桌面图标即可同步！**

---

### 方式 3：配置别名

```bash
# 添加到 .bashrc
echo 'alias quiz-sync="/home/bao/.openclaw/workspace/apps/quiz-app/trigger-sync.sh"' >> ~/.bashrc
source ~/.bashrc

# 之后只需执行
quiz-sync
```
