### 题 300/330：手性碳原子判断

**题目**: 下列化合物中含有两个手性碳原子的是？

**选项**:
- A. CH₂OH-CHOH-CHO
- B. CH₂Br-CHBr-Cl
- **C. CH₂OH-CHOH-COOH** ✓
- D. C₆H₅-C(OH)(CH₃)-CH₂Cl

**你的答案**: C ✓  
**正确答案**: C

**解析**: 手性碳是连 4 个不同基团的饱和碳原子。C 选项有 2 个手性碳。

**知识点**: 手性碳原子判断

- A. CH₂OH-CHOH-CHO

### 题 332：N₁₃⁺离子结构分析

**题目**: 理论化学模拟得到一种 N₁₃⁺离子，结构如图（中心 N 连接 4 个 N₃基团）。下列说法**错误**的是？

**选项**:
- A. 所有原子均满足 8 电子结构
- **B. N 原子的杂化方式有 2 种** ✓

### 题 333：配合物结构分析

**题目**: 铁的配合物 [Fe(Htrz)₃](ClO₄)₂，其中 Htrz 为 1,2,4-三氮唑。下列说法**错误**的是？

**选项**:
- A. 阴离子 ClO₄⁻空间构型为正四面体
- B. Htrz 分子中 σ 键数为 6 个
- **C. 与 Fe²⁺形成配位键的原子为 N** ✓
- D. 空间结构为八面体

**你的答案**: 先 A 后 D  
**正确答案**: B

**解析**: 
- ClO₄⁻：正四面体 ✓
- Htrz（三氮唑）：5 元环 5 个 σ 键 + 3 个 N-H 键 = **8 个 σ 键**
- N 原子有孤电子对，与 Fe²⁺配位 ✓

**你的错误**: σ 键数答 6 个（应为 8 个）

### 题 211：机械波综合计算（两列波相向传播）

**题目**: 两列简谐横波分别沿 x 轴正负方向传播，波源在 x=-0.2m 和 x=1.2m，波速 v=0.4m/s，振幅 2cm。t=0 时 P(0.2m)、Q(0.8m) 刚开始振动，M 在 x=0.5m。求 M 点的振动情况？

**选项**:
- A. 振幅 2cm
- **B. 振幅 4cm** ✓
- C. 始终静止
- D. 振幅 0

**你的答案**: B ✓  
**正确答案**: B

**解析**: M 到两波源距离相等（都是 0.7m），两列波同时到达，相位相同，振动加强，振幅=2+2=4cm。

**知识点**: 波的叠加原理

### 题 212：B 超超声波图像分析

**题目**: B 超检测仪超声波在人体内传播速度 1200m/s，波长λ=8mm。下列说法**正确**的是？

**选项**:
- A. 频率为 1.2×10⁵Hz
- B. 质点 B 在 1×10⁻⁴s 内路程为 0.12m
- **C. 质点 A 此时沿 y 轴正方向运动** ✓
- **D. A、B 两点加速度大小相等、方向相同** ✓

**你的答案**: CD ✓  
**正确答案**: CD

**解析**: 
- A 错：f=v/λ=1200/0.008=1.5×10⁵Hz
- B 错：路程=15×4A=0.24m
- C 对：波沿 x 正方向传播，A 点向上运动
- D 对：A、B 都在波峰，加速度相同

**知识点**: 波的频率计算、质点振动方向、加速度分析

### 题 213：机械波综合计算（起振方向、位移、路程）

**题目**: 两列简谐横波相向传播，波速 v=0.4m/s，振幅 2cm。t=0 时波形如图。求：(1)M 的起振方向 (2)t=2s 时 M 的纵坐标 (3)0-2s 内 M 的路程

**选项**:
- A. 起振向下，y=-2cm，S=16cm
- B. 起振向上，y=2cm，S=20cm
- **C. 起振向下，y=-4cm，S=20cm** ✓
- D. 起振向上，y=-2cm，S=16cm

**你的答案**: C（修正后）✓  
**正确答案**: C

**解析**: 
1. 两波源起振方向都向下，M 起振向下
2. T=λ/v=1s，t=2s=2T，M 在波谷。这是振动加强点，振幅=2A=4cm，所以 y=-4cm
3. M 从 t=0.75s 开始振动，振动时间 1.25s=5T/4，路程=5×4cm=20cm

**你的错误**: 第二问纵坐标答 -2cm（应为 -4cm，振动加强点振幅是 4cm）

**知识点**: 波的叠加、振动加强点、路程计算

### 题 405：动物细胞培养（肝肿瘤 vs 正常细胞）

**题目**: 关于肝肿瘤细胞（甲）和正常肝细胞（乙）的培养，叙述**正确**的是？

**选项**:
- A. 制备肝细胞悬液用胃蛋白酶处理
- B. 通入 5%CO₂刺激细胞呼吸
- C. 加入干扰素防止杂菌污染
- **D. 原代培养中乙细胞会出现停止增殖** ✓

**你的答案**: D ✓  
**正确答案**: D

**解析**: 
- A 错：用胰蛋白酶或胶原蛋白酶
- B 错：CO₂作用是维持 pH
- C 错：加抗生素防污染
- D 对：正常细胞有接触抑制

**知识点**: 动物细胞培养条件、接触抑制

### 题 409：愈伤组织应用

**题目**: 愈伤组织中的细胞全能性很强，在植物细胞工程中应用广泛。愈伤组织可用于？

**选项**:
- A. 在 NAA/6-BA 比值高的培养基中诱导形成芽
- B. 利用离体诱变技术定向获得所需优良植株
- **C. 通过酶解法去除细胞壁获得球形的原生质体** ✓
- D. 悬浮振荡培养获得细胞系后直接投入发酵生产

**你的答案**: 先 B 后 C ✓  
**正确答案**: C

**解析**: 
- A 错：NAA/6-BA 比值高诱导生根，比值低诱导生芽
- B 错：诱变是不定向的
- C 对：愈伤组织细胞壁薄，可用酶解法获得原生质体
- D 错：植物细胞培养不能直接发酵

**知识点**: 植物激素配比、原生质体制备

### 题 410：克隆猴技术

**题目**: 科学家培育克隆猴的基本步骤，下列相关叙述**不正确**的是？

**选项**:
- **A. 动物细胞核移植技术，体现了动物细胞的全能性** ✓
- B. 对 A 猴卵母细胞去核是为了保证克隆猴的核遗传物质都来自 B 猴
- C. 必须将重组细胞培育到一定程度后才能植入代孕动物体内
- D. A 猴的卵母细胞的细胞质中含有促进 B 猴体细胞核全能性表达的物质

**你的答案**: 先 D 后 A ✓  
**正确答案**: A

**解析**: 
- A 错：体现的是**动物细胞核的全能性**，不是细胞全能性
- B 对：去核保证核遗传物质来自供体
- C 对：需培育到早期胚胎才能移植
- D 对：卵母细胞质中含有促进核全能性表达的物质

**知识点**: 动物细胞核移植、细胞核全能性
